# NonDominant

JoyCon を左手デバイスとして使うためのソフトウェア


# 必要な環境

- Windows 10 以降 64 bit
- .NET 6 Runtime
- JoyCon
- Bluetooth レシーバー

# 簡単な使い方

NonDominant.exe を起動します。

Bluetooth レシーバーで JoyCon を認識させてください。
SL と SR の間のボタンをあれこれすると認識されます！

あとは、NonDominant で適当なショートカットを設定してください。
ZL と ZR は押しながらほかのボタンを押すと機能が発動するシフトキーのような扱いになっています。
（ダウンやホールドを使うと ZL/ZR 自体に機能を割り当てることもできます。）

- ダウン：ボタンを押した瞬間発動
- ホールド：押している間発動
- クリック：ボタンを押してすぐ離すと発動（ダブルクリックを割り当てているときは、そこまで待ちます）
- アップ：ボタンを離したら発動
- ダブルクリック：ボタンを押してすぐ離すを2回すると発動
